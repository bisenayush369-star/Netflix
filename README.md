# 🎬 Netflix Landing Page Clone

A responsive **Netflix India landing page clone** built using **HTML & CSS**.  
This project focuses on modern UI design, layout accuracy, and responsiveness inspired by the official Netflix website.

---

## 🚀 Live Preview
https://netflex-ayushdevx.netlify.app/

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- Google Fonts
- Flexbox & Grid
- Responsive Design (Media Queries)

---

## ✨ Features
- Fully responsive layout (desktop & mobile)
- Netflix-style hero section
- Embedded videos inside TV frames
- Clean FAQ section UI
- Modern footer layout
- Pure HTML & CSS (no frameworks)
